# 学園アイドルマスター メモリー編成共有サイト

学マスのステージメモリー編成を投稿・検索・共有できるWebサイトです。

## 🚀 主な機能

- メモリー編成の投稿・編集
- ステージ・アイドル別検索
- 編成詳細の閲覧
- Discord OAuth2 ログイン（SuperTokens セッション管理）

## 🛠 技術スタック

### フロントエンド
- React 18
- TypeScript
- Vite
- Tailwind CSS

### バックエンド
- Python
- FastAPI
- MongoDB
- Authlib + Discord OAuth2
- SuperTokens (セッション管理)

### インフラ
- Vercel (フロントエンド)
- Railway (バックエンド)

## 📦 開発環境セットアップ

### 必要な環境
- Node.js 18+
- Python 3.11+
- MongoDB
